# 🪐 Solar Portfolio

Interactive creative portfolio with a solar system theme. Each planet represents a portfolio section: profile, skills, projects, experience, contact, and more.

Built with **Astro 5** + **TypeScript** + **vanilla CSS**. No external frontend dependencies.

## Tech Stack

- [Astro](https://astro.build) v5 — static site generator
- TypeScript — strict mode
- Vanilla CSS — animations, custom properties, responsive
- Native `<dialog>` — modal without libraries

## Solar System

| Planet      | Section               |
|-------------|-----------------------|
| ☀️ Sun      | Martín (identity)     |
| Mercury     | Skills                |
| Venus       | About me              |
| 🌍 Earth    | Education             |
| 🔴 Mars     | Featured projects     |
| Jupiter     | Experience            |
| Saturn      | Contact               |
| Uranus      | Reflection            |
| Neptune     | Extras                |

Click any planet to open the modal with detailed info.

## Development

```bash
npm install
npm run dev       # http://localhost:4321
npm run build     # output in dist/
npm run preview   # preview production build
```
